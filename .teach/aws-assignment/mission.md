# Mission: AWS Assignment

## Why

The assignment gives optional additional-feature credit for AWS hosting and cloud security services. The goal is to know exactly what to build, what to screenshot, and how to explain it in the report or presentation.

## Success looks like

- Explain the AWS architecture for this project in one minute.
- Host the frontend on S3 or describe the exact S3 deployment steps confidently.
- Move secrets out of code into AWS Secrets Manager conceptually and show what evidence proves it.
- Explain where Cognito fits and why it replaces custom login storage for a production version.

## Constraints

- Free hosting only where possible.
- Keep the lesson focused on the assignment, not full AWS operations.
- Do not publish real secrets, passwords, access keys, or JWT signing values in screenshots.

## Out of scope

- Enterprise AWS networking.
- Paid production scaling.
- Rewriting the full project to use Cognito end to end.
